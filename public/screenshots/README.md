# GovernanceOS Screenshots

These HTML files render pixel-perfect mockups of each major screen.

**To create PNG screenshots for the GitHub README:**

1. Open each `.html` file in Chrome
2. Press `Cmd+Shift+P` → type "Screenshot" → select "Capture full size screenshot"
3. Save the PNG to this folder with the same base name

| File | Page | Recommended size |
|------|------|-----------------|
| screenshot-dashboard.html | Dashboard overview | 1280×800 |
| screenshot-entity-registry.html | Entity registry | 1280×800 |
| screenshot-board-meetings.html | Board meetings | 1280×800 |
| screenshot-tor-generator.html | Terms of Reference | 1280×800 |
